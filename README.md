# Authenticator FE

Authenticator is a login register page built on Hapi.JS and ReactJS. The authentication is performed using JWT auth tokens <br />

## Tech Stack
- BE - Hapi.JS 
- FE - ReactJS
- Authentication: JWT

## Usage
```
git clone https://github.com/siddharth-lakhara/authenticator-FE.git
```
